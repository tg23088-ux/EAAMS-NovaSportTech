## About This Project
EAAMS is a centralized software platform designed for sports training academies to:
- Track and manage player performance metrics over time
- Manage tournament registrations, brackets, and schedules
- Provide role-based dashboards for Admins, Coaches, and Players

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MySQL (via phpMyAdmin)
